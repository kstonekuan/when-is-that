# When Is That?

A timezone comparison tool that displays two side-by-side analog clocks with calendars, making it easy to see what time it is (or will be) in different timezones.

## Features

- **Side-by-side clocks** — Compare two timezones with analog clock faces and calendars
- **Live mode** — Clocks tick in real time
- **Custom mode** — Set a specific date and time to see the conversion (toggle with Space)
- **Timezone search** — Search and select from all IANA timezones with abbreviation display
- **Auto-detection** — Your local timezone is detected automatically

## Getting Started

```sh
pnpm install
pnpm dev
```
